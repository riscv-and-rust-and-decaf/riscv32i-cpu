# Flash 速度测试

## 结论
在 50 MHz 时钟下, 某个周期给出的读 Flash 请求, 在下下个周期能够得到答案.

```
      +--------+        +--------+         +--------+        +--------+         +--------+
clk   |        |        |        |         |        |        |        |         |        |
     -+        +--------+        +---------+        +--------+        +---------+        +-------

                        |     cycle 1      |     cycle 2     |     cycle 3      |

    -----------------------\ /------------------------------------------------\ /----------------
addr   xxxxx                X    addr                                          X   xxxxx
    -----------------------/ \------------------------------------------------/ \----------------

                        |                  |                 |                  |

    ------------------------------------------------------------------\ /------------------------
data              xxxxxxxxxxx                                          X      data
    ------------------------------------------------------------------/ \------------------------
```

## 实验
实验很简单, 直接看源代码即可.
